## 📐 Espacements

- **Sections principales** : `py-24` (padding vertical 96px)
- **Cartes** : `p-6` ou `p-8` (padding 24px ou 32px)
- **Gaps** : `gap-6` ou `gap-8` (24px ou 32px entre éléments)
- **Marges bottom** : `mb-16` (64px entre titres et contenu)

## 🔤 Typographie

- **Titres de sections** : `text-4xl md:text-5xl font-bold`
- **Sous-titres** : `text-xl text-slate-600`
- **Titres de cartes** : `text-xl font-bold text-slate-900`
- **Texte body** : `text-slate-600`

## ✨ Animations

Toutes les sections utilisent Framer Motion :
- **initial** : `{ opacity: 0, y: 30 }`
- **whileInView** : `{ opacity: 1, y: 0 }`
- **transition** : `{ duration: 0.5-0.8 }`
- **viewport** : `{ once: true }`

## 🎯 Points Clés

1. **Suppression des statistiques** ✅
2. **Explication "How it works"** ✅ (4 étapes + "What is FlyNova")
3. **Présentation ACARS** ✅ (Section complète dark theme)
4. **VAs améliorées** ✅ (6 VAs max, bouton "View All")
5. **Section donation professionnelle** ✅ (3 valeurs + bloc donation)
6. **Footer complet** ✅ (Liens, réseaux, copyright)

## 📱 Responsive

- **Mobile** : 1 colonne
- **Tablet (md)** : 2 colonnes
- **Desktop (lg)** : 3-4 colonnes

Tous les éléments s'adaptent automatiquement grâce aux classes Tailwind.
